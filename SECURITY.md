# Security Policy

## Reporting a Vulnerability

To report a security vulnerability in this repository, please email **security@3leaps.net**.

Do not open a public GitHub issue for security vulnerabilities.

## Response Timeline

We aim to acknowledge reports within 48 hours and provide an initial assessment within 7 days.

## Scope

This repository contains documentation and standards only—no runtime code. Security concerns are most likely to involve:

- Incorrect security guidance in documentation
- Patterns that could lead to vulnerabilities if followed

## More Information

See [3leaps/oss-policies](https://github.com/3leaps/oss-policies) for complete security policies.

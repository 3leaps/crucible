# Release v0.1.22

**A governance release: EPR-0001 graduates to accepted.**

## Overview

v0.1.22 graduates the inaugural Engineering Principle Record, EPR-0001, from
proposed to accepted. No schema changes, no capability-token bump, no contract
movement — a status change plus the reference implementation that earns it.

## Highlights

### EPR-0001: proposed → accepted

EPR-0001 states that every published artifact built from a resolved dependency
graph must ship that graph pinned in-repo, enforced at build, continuously
audited, and held at parity across all distribution surfaces of the same release.
It has stood as **proposed** pending a conforming reference implementation — the
record's own graduation condition.

That implementation has now landed publicly. The first conforming adopter ships a
shared cryptographic core through two published surfaces — a CLI binary and a
native language addon, the exact "one core, many bindings" shape the parity
obligation governs — and discharges all four obligations:

- **Pin** — each surface commits its resolved graph in-repo.
- **Enforce** — the artifact builds consume the pins in locked mode, with a guard
  proving no published-artifact build path can drop the locked flag.
- **Audit** — every pinned surface is scanned for advisories on change and on a
  schedule.
- **Parity** — a declared, reviewed parity manifest asserts the security-critical
  shared components resolve to an identical upstream identity across both
  surfaces; divergence fails the build.

Crucially, pin, enforce, and parity are each demonstrated by an **executable
negative control** — the gate is shown to _fail_ on a violated or stale pin, a
bypassing build path, and a divergent, absent, or undeclared parity entry — while
audit is demonstrated by on-change and scheduled advisory scans of every committed
lockfile, not merely asserted present in CI. That is the standard EPR-0001 sets for
conformance, and it is the standard the reference implementation meets.

The record gains a **Reference implementation** section linking that public
conformance work, and its status is now **accepted**.
